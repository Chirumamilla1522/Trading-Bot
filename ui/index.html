AI AGENTS · DATA FLOW

All agents read and write the same FirmState: market data, news, risk, convictions, and the XAI reasoning log. Tier 1 and Tier 2 run in the background; Tier 3 is one LangGraph pass when triggered.

Loading MLflow status…

T1 Always on · lightweight

MovementTracker (~30s) updates price/vol + non-news bias. SentimentMonitor (~60s) synthesizes one desk score from Tier-2 structured news (SQLite / memory) with an LLM-first path + deterministic fallback. Strong signals help fire Tier 3.

T2 Periodic

News ingestion runs continuously (Benzinga + yfinance). Each headline is scored locally with FinBERT (sentiment/confidence) and a heuristic impact model (impact_score, urgency tier, vol-prob), then pushed into a shared NewsPriorityQueue. NewsProcessor (~2m) drains the queue top-first (T0/T1 before tail), enriches with LLM analysis (category, themes, affected tickers), and writes structured rows + cross-stock impacts. FundamentalsRefresher (~4h) refreshes snapshots and flags material changes. Options chains refresh on a separate ~15s API loop.

T3 Triggered pipeline

Manual run, auto T1 thresholds, scanner, or cooldown-aware timer. Tier 3 is one LangGraph pass; it consumes Top-K priority news (queue) + Tier-2 digests, then appends reasoning and mutates state.
